# OUxInfo
OUxInfo is a C++ based fast Shannon entropy estimator for Python.

## Dependencies
* Python (>=3.12 is better)
  * pybind11 (will be installed automatically)
* gcc >= 11 (14 is better)
  * Boost
    ~~~bash
    $ sudo apt install libboost-dev
    ~~~
  * nanoflann (included in this repository)

## Setup
1. Compile OUxInfo
    ~~~bash
    $ pip install .
    ~~~
